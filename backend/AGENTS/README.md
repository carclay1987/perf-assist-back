# AGENTS

Раздел для всех md-файлов, описывающих агентов и их правила работы с проектом Perf Assist.

План:
- `backend-architect.md` — агент для проектирования и эволюции Go-бэкенда.
- `frontend-architect.md` — агент для архитектуры фронтенда и работы с Figma/design-проектом.
- `llm-orchestrator.md` — агент для проектирования LLM-промптов и цепочек.
- `product-owner.md` — агент, следящий за соответствием фич продуктовой спецификации (`product/*`).
- `fullstack-integrator.md` — агент, который следит за согласованностью API между фронтом и бэком.

Далее мы добавим сюда описание проекта и детализированную архитектуру бэкенда Perf Assist.
